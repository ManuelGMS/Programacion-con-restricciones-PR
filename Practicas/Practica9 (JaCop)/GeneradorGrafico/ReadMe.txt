Para generar las imágenes de la traza son necesarios los
archivos "tree.xml" y "vis.xml" que se generan automáticamente
cuando se ejecuta un programa JaCop haciendo uso de "TraceGenerator".



Una vez se tienen los dos archivos, estos se pondrán junto al archivo
"config.xml" en este mismo directorio.



Ahora que ya tenemos los tres archivos solamente será necesario ejecutar
el archivo "generate" ( .bat --> Windows / .sh --> Linux ), si la ejecución
ha sido correcta la salida debería encontrarse en la carpeta "output".
